Vocab:

  1. Sprite - A graphics object that can be moved on top of or behind a background.

  2. Script - A set of instructions. In Scratch a script is a single stack of blocks belonging to one sprite or to the stage.

  3. Iteration - Repeating a step-by-step procedure several times; also used to refer to single execution of that repetitive procedure.

  4. Integrated Development Environment (IDE) - Provides a developer with a way to create a program, run the program, and debug the program all within one application.

  5. Boolean Expression - Evaluates to either true or false; used in the conditional of an if-structure.

  6. Execute - To run a program or a single instruction.

  7. Bug - Part of a program that causes an error or undesired output.

  8. Debugging - The process of figuring out why code doesn't behave as expected and eliminating bugs to make it work as desired.

  9. Central Processing Unit (CPU, or core) - The part of the computer that executes one instruction at a time.

  10. Packet - A chunk of data transmitted as a unit over a network, sometimes in smaller bursts called frames.

  11. Router - A piece of hardware responsible for dictating the flow of packets over a network.

  12. ISP - Internet Service Providers make Internet access available to users.

  13. Protocol - An agreed-upon method for conducting a communication.

  14. IPv4 - The Internet Protocol version 4 is the dominant protocol for routing traffic on the Internet, specifying to and from addresses using a dotted decimal such as 122.45.255.0 dotted decimal notation - A representation of numerical data in which groups of base-10 numbers are separated by dots.

  15. IPv6 - The Internet Protocol version 6 provides a large number of new addresses to route Internet traffic, using from and to addresses written as colon-hexadecimal notation, such as fe80::42:acff:feaa:1bf0.

  16. Domain Name System (DNS) - A hierarchical system for naming resources on the Internet.

  17. name server - A computer dedicated to providing responses to requests for Domain Name information.
