Vocab:

  1. Sprite - A graphics object that can be moved on top of or behind a background.

  2. Script - A set of instructions. In Scratch a script is a single stack of blocks belonging to one sprite or to the stage.

  3. Iteration - Repeating a step-by-step procedure several times; also used to refer to single execution of that repetitive procedure.

  4. Integrated Development Environment (IDE) - Provides a developer with a way to create a program, run the program, and debug the program all within one application.

  5. Boolean Expression - Evaluates to either true or false; used in the conditional of an if-structure.

  6. Execute - To run a program or a single instruction.

  7. Bug - Part of a program that causes an error or undesired output.

  8. Debugging - The process of figuring out why code doesn't behave as expected and eliminating bugs to make it work as desired.

  9. Central Processing Unit (CPU, or core) - The part of the computer that executes one instruction at a time.

  10. Packet - A chunk of data transmitted as a unit over a network, sometimes in smaller bursts called frames.

  11. Router - A piece of hardware responsible for dictating the flow of packets over a network.

  12. ISP - Internet Service Providers make Internet access available to users.

  13. Protocol - An agreed-upon method for conducting a communication.

  14. IPv4 - The Internet Protocol version 4 is the dominant protocol for routing traffic on the Internet, specifying to and from addresses using a dotted decimal such as 122.45.255.0

  15. dotted decimal notation - A representation of numerical data in which groups of base-10 numbers are separated by dots.

  16. IPv6 - The Internet Protocol version 6 provides a large number of new addresses to route Internet traffic, using from and to addresses written as colon-hexadecimal notation, such as fe80::42:acff:feaa:1bf0.

  17. Domain Name System (DNS) - A hierarchical system for naming resources on the Internet.

  18. name server - A computer dedicated to providing responses to requests for Domain Name information.

  19. browser tab - A separate viewing pane within a browser window, many of which may be open at once, allowing a user to switch quickly between web pages.

  20. bookmarks - A piece of data used by a browser to allow the user to return quickly to a previously visited site.

  21. history - Information stored by a browser about the web resources that it has requested.

  22. browser - A piece of software that renders web pages and allows users to view resources on a network.

  23. client application - A program that makes use of another piece of software or set of data; for web interactions, the web browser is usually the client application.

  24. client machine - A machine running a client application.

  25. TCP/IP - The dominant set of protocols for the transfer of data over the Internet.

  26. redundancy - The property of having many resources or components which all serve identical purposes.

  27. URL - Uniform Resource Locators are strings of characters that contain information about how to access a resource over the Internet.

  28. parameters - A value used in a model

  29. port - A subset of network traffic received at an IP address, specified by a port number. Traffic at any particular port number is usually listened to by a particular type of application or process that is being accessed on a server.

  30. filter - A program or device that allows or passes some packets through while blocking others based on predetermined criteria.

  31. cookie - A small piece of data stored on the client machine that can only be accessed by the server that issued it.

  32. Add-ons - A piece of software designed to enhance the capabilities of another piece of software.

  33. third-party cookies - A cookie that is stored on a client computer by a domain other than the one that is currently requesting resources.

  34. HTML - HyperText Markup Language is the most common language used for creating web pages.

  35. CSS - Cascading Style Sheets contain hierarchical information about how the content of a web page will be rendered in a browser.

  36. JavaScript - A client-side scripting language used to write programs that are embedded in web pages, often to make a web page more interactive.

  37. web crawler - A softbot responsible for following hyperlinks throughout the Internet to provide information for the creation of a web index.

  38. web index - A database that stores information about web content.

  39. query - A request for information.
